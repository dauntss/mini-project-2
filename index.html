CSS Snippet Cheatsheet

Ever have trouble recalling the exact syntax for your favorite CSS code? Give it a permanent home and add it to this page! Select any snippet below and it'll automatically select all of the code you copy.

Flexbox Row

Use these three properties to create a Flexbox row layout.

.flex {
  display:flex;
  flex-direction:row;
  flex-wrap:wrap;
}
Flexbox Column

Use this to create a Flexbox column layout.

.column {
  display:flex;
  flex-direction:column;
}
CSS Grid Layout

Build a 12-column input using CSS Grid.

.grid {
  display: grid;
  width: 100%;
  grid-template-columns: repeat(12, 1fr);
}
Linear Gradients

This will create a background linear-gradient from top left to bottom right.

.linear-gradient-background {
  background: linear-gradient(to bottom right, lightorange, pink, deeppink);
}
Box Transition Glow

Use transition and box shadow to glow on hover.

.code-card .card-header {
  border-radius:8px;
  transition:all 0.5s ease-in-out;
}

.code-card:hover, .code-card:hover .code-header {
    box shadow:inset 0 0 8px purple, 0 0 15px purple;
}
Overlay Card with Title

Use position properties and negative margins to raise elements higher than their natural starting point.

.card-header {
  position:telative;
  margin-top:-20px;
}
